HTML pages Timestamps graphic representations requested to a JSF servlet in a tomcat app server. The image below gives a picture of all the pages requested to a given URL . The root page will display a bar for every page that has been requested for the URL

JSF server configuration

Set environmental variable:

LBHANDSHAKE=http://ourfamilycontacts.com/loadbalancer/loadinstance/includes/js/lbhandshake.js
Test:
LBHANDSHAKE=http://10.0.0.246:8080/loadbalancer/loadinstance/includes/js/lbhandshake.js

faces-config.xml

<lifecycle>
<phase-listener>com.assoc.jad.loadbalancer.lbinstance.LifeCycleListener</phase-listener>
</lifecycle>

Web.xml

<filter>
<filter-name>jsftrace filter</filter-name>
<filter-class>com.assoc.jad.loadbalancer.lbinstance.LoadBalancerFilter</filter-class>
</filter>
<filter-mapping>
<filter-name>jsftrace filter</filter-name>
<servlet-name>Faces Servlet</servlet-name>
<url-pattern>/*</url-pattern>
</filter-mapping>

JVM:

--add-opens jdk.management/com.sun.management.internal=ALL-UNNAMED

Download jar file

Jar-file: /jadallfolders/jadtemp/Loadbalancer.jar
